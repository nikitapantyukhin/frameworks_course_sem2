# Лабораторная работа №1: 
## Описание проекта
Данная работа посвящена анализу влияния образа жизни студентов на их академическую успеваемость (CGPA) и психическое здоровье (риск депрессии). В ходе работы реализован полный цикл разработки модели машинного обучения: от первичного анализа данных до оценки качества и интерпретации результатов.

## Что было реализовано

### 1. Расширенный визуальный анализ (EDA)
- Построена **матрица корреляций** (Heatmap) для выявления скрытых связей между привычками студентов.
- Использованы **скрипичные графики (Violin Plots)** для детального сравнения времени сна и учебы у разных групп.
- Проанализирован дисбаланс целевых классов.

### 2. Предобработка и очистка
- Удаление неинформативных признаков и обработка логических аномалий.
- Кодирование категориальных переменных (One-Hot Encoding).
- Масштабирование признаков с помощью **StandardScaler** для корректной работы линейных моделей.

### 3. Прогнозирование успеваемости (Регрессия)
- Обучена модель Линейной регрессии для предсказания `CGPA`.
- Проведен **анализ остатков (Residual Analysis)**, визуализировано распределение ошибок прогноза.

### 4. Выявление риска депрессии (Классификация)
- Обучена Логистическая регрессия.
- **Решена проблема дисбаланса классов** с помощью параметра `class_weight='balanced'`, что позволило значительно поднять метрику **Recall** (способность находить студентов в группе риска).
- Построена **ROC-кривая** и рассчитана метрика **AUC-ROC** для оценки разделяющей способности модели.
